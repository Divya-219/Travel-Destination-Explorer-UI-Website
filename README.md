# 🌍 WanderScope – Travel Destination Explorer

## Project Pitch

WanderScope is a travel-based web application designed for users who want to explore destinations and check important information like weather before planning a trip.

## Problem Statement
Users often need to visit multiple websites to explore destinations and check weather conditions, making travel planning time-consuming and inefficient.

## User Persona
Travel enthusiasts 

## Solution
WanderScope is a dynamic web application that allows users to:
 - Explore travel destinations
 -  Search and filter destinations in real time  
 - View real-time weather data
 -  Send travel inquiries through a contact form

## Features

 -  Dynamic destination gallery (JavaScript)
 -  Real-time search and category filtering system  
 - Weather API integration
 -  Responsive design
 - Contact form with validation

## Tech Stack

    HTML5
    CSS (Flexbox + Grid)
    JavaScript
    Open-Meteo Weather API

## Responsive Design

    Mobile-first layout
    Grid layout changes to single column on small screens
    Optimized navigation for all devices

## Live Data (API)

Weather data is fetched using: https://open-meteo.com/
Form Validation

## Contact form includes:

   The contact form includes:

- Required field validation  
- Email format validation using regex  
- Custom error messages (no default alerts)  


## Challenges & Solutions
One challenge was managing multiple JavaScript features such as filtering, search functionality, and weather API integration without code conflicts or duplicated logic.
This was solved by splitting the code into reusable functions like renderCards(), applyFilters(), and getWeather(), improving structure and maintainability.

##  Deployment
The project is deployed using GitHub Pages and Vercel.
